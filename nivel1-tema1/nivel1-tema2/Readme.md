# Ejercico hacer de la clase Tigger un Patrón Singleton

Este ejercicio muestra cómo implementar el patrón de diseño Singleton en PHP, utilizando al personaje Tigger de 
"Winnie the Pooh".
El patrón Singleton garantiza que solo exista una única instancia de la clase durante toda la ejecución del 
programa.
LEn la clase Tigger creo un constructor privado, por lo que no se puede crear directamente con new.
